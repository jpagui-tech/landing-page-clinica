# landing-page-clinica

Site estático para uma clínica de saúde usando Tailwind CSS via CDN.

## Arquivos principais

- `index.html` — página principal com layout responsivo e formulário.

## Como usar

1. Abra `index.html` no navegador.
2. Ou use um servidor local, por exemplo:

   ```bash
   npx serve .
   ```

3. Personalize textos, imagens e dados de contato conforme necessário.
